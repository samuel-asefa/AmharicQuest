<h2 align="center"><em><code>AmharicQuest</code></em></h2>
An Amharic learning app created with Next.js, Firebase, and Framer.

## Features
- Fidel Explorer
- Structured Learning Path
- Vocabulary Quiz
- XP & Leveling System

---

## Content

### Fidel
All 33 Ge'ez script consonant families, each with their 7 vowel orders (ä, u, i, a, ē, ə, o), totaling 230+ characters with phonetic romanization.

### Vocabulary
| Category | Words |
|---|---|
| Greetings | 4 |
| Basic | 2 |
| Family | 2 |
| Numbers | 10 |
| Food | 3 |
| Verbs | 6 |
| Adjectives | 4 |
| Time | 4 |
| Objects & Animals | 7 |

### Lessons
9 structured lessons from beginner to intermediate, each rewarding 30–80 XP on completion.
